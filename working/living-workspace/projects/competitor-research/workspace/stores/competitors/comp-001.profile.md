# Linear — Competitor Profile

## One-liner
The fast, opinionated issue tracker for software teams. Built around speed, keyboard-first UX, and developer aesthetics.

## Positioning
Premium issue tracker. Targets engineering-led product teams that value craft and speed over flexibility. Explicitly opinionated — refuses to be a generic project tool.

## Product strengths
- Best-in-class performance and keyboard UX
- Highly opinionated workflows that just-work for software teams
- Clean, minimal aesthetic that engineers love
- Strong API and integration ecosystem

## Product gaps (relative to broader project planning)
- Limited support for non-engineering work (marketing campaigns, ops projects)
- Less flexible than tools like ClickUp or Asana for cross-functional coordination
- Custom workflows are constrained by the opinionated model

## What we'd compete on
- Flexibility for non-engineering projects (marketing, ops, content)
- Cross-functional visibility — Linear assumes engineering audience
- Pricing for non-engineering teams
