Three column-add modes

  • Computed — formula over existing fields. View-side, no data write. The age_days column above.
  • Enriched — Claude walks each instance and writes a value. Persistent. Example: "add a topic column and classify each task".
  • Manual — empty new field; you fill in yourself. Becomes part of the contract.
